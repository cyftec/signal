Common Patterns

Frequently used patterns and techniques when working with signals.

Array Mutations

Array signals provide mutation methods that trigger effects:

const items = signal([1, 2, 3]);

items.push(4);          // Add item
items.pop();            // Remove last item
items.splice(1, 2);     // Remove items at index
items.remove((item) => item % 2 === 0); // Remove by predicate

Object Partial Updates

Object signals support partial updates with set():

const user = signal({ name: "John", age: 30 });

user.set({ age: 31 }); // Shallow merge
// user.value is { name: "John", age: 31 }

Type-Specific Traps

Use traps to get type-specific methods for common operations:

import { trap } from "@cyftech/signal";

const count = signal(5);
const trapped = trap(count);

// Number-specific operations
const doubled = trapped.mul(2).result;
const clamped = trapped.toConfined(0, 10).result;

String Operations

String traps provide convenient string manipulation:

const name = signal("john doe");
const trapped = trap(name);

const upper = trapped.UPPERCASE.value;     // "JOHN DOE"
const title = trapped.TitleCase.value;    // "John Doe"
const lower = trapped.lowercase.value;    // "john doe"

Array Operations

Array traps provide array manipulation methods:

const items = signal([1, 2, 3, 4, 5]);
const trapped = trap(items);

const length = trapped.length.value;       // 5
const last = trapped.lastItem.value;      // 5
const reversed = trapped.reversed.value;  // [5, 4, 3, 2, 1]

// Partition array
const [evens, odds] = trapped.partition((item) => item % 2 === 0);
// evens.value is [2, 4], odds.value is [1, 3, 5]

Logical Operations

Use operations for complex logical and mathematical chains:

import { op } from "@cyftech/signal";

const count = signal(5);

const result = op(count)
  .add(3)
  .mul(2)
  .isBetween(10, 20)
  .truthy.value;

Signal Connections

Connect signals together with connectors:

import { transmit, receive } from "@cyftech/signal";

// Broadcast to multiple receivers
const source = signal("hello");
const receiver1 = signal("");
const receiver2 = signal("");

const effect = transmit(source, receiver1, receiver2);
source.value = "hi"; // Both receivers update

// Connect multiple transmitters to one receiver
const transmitter1 = signal("a");
const transmitter2 = signal("b");
const receiver = signal("");

const effects = receive(receiver, transmitter1, transmitter2);

Template Interpolation

Use template literals with signals:

import { tmpl } from "@cyftech/signal";

const firstName = signal("John");
const lastName = signal("Doe");

const fullName = tmpl`${firstName} ${lastName}`;
console.log(fullName.value); // "John Doe"

Promise States

Manage async operations with promise states:

import { promstates } from "@cyftech/signal";

const fetchData = async (id: number) => {
  const response = await fetch(`/api/data/${id}`);
  return response.json();
};

const [runPromise, result, error, isRunning] = promstates(fetchData, null);

await runPromise(1);
console.log(result.value);    // Fetched data
console.log(error.value);    // undefined
console.log(isRunning.value); // false

Function-Based Derived Signals

Use compute() for function-based derived signals:

import { compute } from "@cyftech/signal";

const a = signal(5);
const b = signal(3);

const sum = compute((x: number, y: number) => x + y, a, b);
console.log(sum.value); // 8
