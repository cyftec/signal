First Example

Let's build a simple traffic light controller to understand how signals work in practice.

The Problem

We want to create a traffic light that cycles through green, yellow, and red states every 10 seconds, and display appropriate messages to users.

Step 1: Create the Signal

First, create a signal to track the current traffic light color:

import { signal } from "@cyftech/signal";

const color = signal("green");

Step 2: Cycle the Colors

Set up an interval to cycle through the colors:

const TRAFFIC_LIGHT_CHANGE_CUTOFF_IN_MS = 10000;

setInterval(() => {
  if (color.value === "green") color.value = "yellow";
  if (color.value === "yellow") color.value = "red";
  if (color.value === "red") color.value = "green";
}, TRAFFIC_LIGHT_CHANGE_CUTOFF_IN_MS);

Step 3: React to Changes

Use an effect to display messages when the color changes:

import { effect } from "@cyftech/signal";

effect(() => {
  if (color.value === "green")
    updateUiWithMessage("Keep moving. Don't congest the traffic.");

  if (color.value === "yellow")
    updateUiWithMessage("Slow down! Signal is about to stop.");

  if (color.value === "red")
    updateUiWithMessage("STOP. Please do not cross the crossing.");
});

Complete Example

Here's the complete code:

import { signal, effect } from "@cyftech/signal";

const color = signal("green");
const TRAFFIC_LIGHT_CHANGE_CUTOFF_IN_MS = 10000;

setInterval(() => {
  if (color.value === "green") color.value = "yellow";
  if (color.value === "yellow") color.value = "red";
  if (color.value === "red") color.value = "green";
}, TRAFFIC_LIGHT_CHANGE_CUTOFF_IN_MS);

// the callback in effect method gets executed every time the value of 'color' signal changes
effect(() => {
  if (color.value === "green")
    updateUiWithMessage("Keep moving. Don't congest the traffic.");

  if (color.value === "yellow")
    updateUiWithMessage("Slow down! Signal is about to stop.");

  if (color.value === "red")
    updateUiWithMessage("STOP. Please do not cross the crossing.");
});

How It Works

  • Signal creation: signal("green") creates a reactive signal with initial value "green"
  • Value updates: Setting color.value triggers all registered effects
  • Effect registration: The effect runs immediately and tracks that it depends on color.value
  • Automatic re-runs: When color.value changes, the effect re-runs automatically

Key Takeaways

Important

Effects must access .value during execution to establish dependencies. If you conditionally access a signal, it won't be tracked if the condition is false on the first run.
