Getting Started

Learn the basics of Signal: installation, creating signals, derived values, and effects.

Installation

Install Signal using Bun:

bun add @cyftech/signal

Core Concepts

Signal provides three core primitives for reactive state management:

  • signal - Creates a mutable source signal from plain values
  • derive - Creates read-only derived signals computed from other signals
  • effect - Registers functions to run when dependent signals change

Creating Signals

Create a signal using the signal() function:

import { signal } from "@cyftech/signal";

const count = signal(0);
const name = signal("John");
const isActive = signal(false);

Access and update signal values using the .value property:

console.log(count.value); // 0
count.value = 1;
console.log(count.value); // 1

Derived Values

Create derived signals that compute values from other signals using derive():

import { signal, derive } from "@cyftech/signal";

const count = signal(0);
const doubled = derive(() => count.value * 2);

console.log(doubled.value); // 0
count.value = 5;
console.log(doubled.value); // 10

Derived signals automatically update when their dependencies change. They are read-only and cannot be directly set.

Effects

Effects run side effects when signals change using effect():

import { signal, effect } from "@cyftech/signal";

const count = signal(0);

effect(() => {
  console.log("Count is:", count.value);
});

count.value = 1; // Logs: "Count is: 1"
count.value = 2; // Logs: "Count is: 2"

Effects run immediately when created and re-run whenever accessed signals change.

Disposal

Clean up effects and derived signals when they're no longer needed:

import { signal, effect, derive, dispose } from "@cyftech/signal";

const count = signal(0);
const doubled = derive(() => count.value * 2);
const eff = effect(() => console.log(count.value));

// Dispose individual items
doubled.dispose();
eff.dispose();

// Or dispose multiple at once
dispose(doubled, eff);
